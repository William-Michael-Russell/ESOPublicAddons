BankerShutUp

=========
What this app does:

Shut the stupid bankers up!
Also quite down crafting tables
Or completely mute the game when talking to a NPC or opening a crafting table


How does it work?
-----------------
Open the bank
Mute the NPC
Close the bank
unmute the npc

Want to change options?
go to the settings menu
